# Cadastro de Produtos
Cadastro de Produtos (Angular 12, API NODE/Json-Server)

## Frontend:
- npm install
- npm install - @angular/cli
- ng s -o

## Backend:

LOCAL
- baixar repositório: https://github.com/ClaudioRogerioTeixeira/produtos-api
- npm start

Remoto
- baixar repositório: https://github.com/ClaudioRogerioTeixeira/produtos-api
- node server.js

## Tecnologias
- Angular
- Angular Material
- Json-Server
- FlexBox
- Flex-Layout
